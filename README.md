# OS Scheduling Algorithms Visualizer

A web-based tool that demonstrates core CPU scheduling algorithms used in operating systems. The website allows users to input processes with arrival time (AT), burst time (BT), and priority, and then runs the algorithms to display:

## First Come First Serve (FCFS)

## Shortest Job First (SJF – non-preemptive)

## Priority Scheduling (non-preemptive)

## Round Robin (RR)

Each algorithm card provides:

Animated Gantt chart for process execution

Result table showing Completion Time (CT), Turnaround Time (TAT), and Waiting Time (WT)

Average TAT and WT for quick analysis

This interactive website is designed for learning, teaching, and visualizing OS scheduling concepts in a practical, hands-on way.
